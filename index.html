Link Converter

Enter either a GitHub repository link or a GitHub Pages link

Examples

GitHub Repository Link:

GitHub Pages Link:
https://username.github.io/repo/path/file.html
Note: The converter works both ways! You can convert from repository links to Pages links and vice versa.
